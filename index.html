      How to use this To Do List Website

      • Create a task using the "Create new task Section" in the "Tasks" tab
      • Click on a Task Created eg."Clean And Arrange House"
      • A Tab for the active task you have created will Appear
      • On the "Add new todo item " Section, ad a to do item eg."Wash utensils", "Wash clothes" etc
      • Click on a todo item once to do item has been completed to mark it
